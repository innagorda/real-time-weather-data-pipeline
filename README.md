# Real-Time Weather Data Pipeline

Проект для изучения новых технологий на практике. Пайплайн сбора, обработки и анализа данных с погодных датчиков (их имитации) в реальном времени с использованием Apache Kafka и ClickHouse.

## Архитектура системы v0.1

```
┌─────────────────┐    ┌──────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│   Датчики       │ -> │   Kafka Cluster  │ -> │   Обработка      │ -> │   ClickHouse    │
│   (Python app)  │    │   (3 ноды KRaft) │    │   Данных         │    │   Аналитика     │
└─────────────────┘    └──────────────────┘    └──────────────────┘    └─────────────────┘
                              │                         │
                              ↓                         ↓
                       ┌─────────────┐           ┌─────────────┐
                       │   Kafka UI  │           │   Grafana   │
                       │  Мониторинг │           │ Визуализация│
                       └─────────────┘           └─────────────┘
```

### Компоненты и версии

- Docker & Docker Compose 4.51.0
- Python 3.10.6
- Apache Kafka 4.1.1
- Clickhouse 25.1.2.3
